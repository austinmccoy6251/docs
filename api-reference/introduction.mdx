## Central API Route

Artemis Search provides a single API route `/fly-search` which is our central search endpoint. You can read more about the route [here](/settings/api-keys). 

## Base URL

All requests contain the following base URL:
```
https://api.search-artemis.com/api
```

## Authentication

All API endpoints are authenticated using Bearer tokens which contain the [API Key](/settings/api-keys).

```
Authorization: Bearer YOUR_ORGANIZATION_API_KEY
```

## Rate Limits

Our central API endpoint `/fly-search` is limited to 30 requests per minute. Please contact us at pallavi@artemisar.com if you need to increase this limit.
